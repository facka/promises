Step 2: Adding reject

A Promise can be rejected, so we are going to add the reject function

then: receives a function that will be executed when the promise is resolved and a function that will be executed when the promise is rejected

reject: receives the reason of why the promises was rejected and call the callback received in the reject function


          var Q = function() {
            var _then, _reject;
            this.then = function(callback, reject) {
                _then = callback;
                _reject = reject;
            };
            this.resolve = function(value) {
                _then(value);
            };
            this.reject = function(error) {
                _reject(error);
            };
          }

Great! now we can manage rejected promises :)
